URL SHORTNER

Create shortned url by

  1. Do An API Call to https://s4o.herokuapp.com/add/<YOUR -URL>
    Example: https://s4o.herokuapp.com/add/http://www.google.com
  2. You will get shortned url as JSON object
    Example: for http://www.google.com corresponding shortned url is
    https://s4o.herokuapp.com/s/4607
  3. Done!
